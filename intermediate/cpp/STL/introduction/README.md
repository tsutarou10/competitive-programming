## string

| プログラム     | 説明                                                              |
| -------------- | ----------------------------------------------------------------- |
| S.substr(l)    | 文字列 S の l 文字目から最後の文字までの部分文字列を返す関数      |
| S.substr(l, r) | 文字列 S の l 文字目から l + r - 1 文字までの部分文字列を返す関数 |
